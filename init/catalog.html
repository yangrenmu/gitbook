目录

目录

Vue.js 的源码是在 src 目录下，目录结构如下：

src
├── compiler        # 编译相关
├── core            # 核心代码
├── platforms       # 不同平台的支持
├── server          # 服务端渲染
├── sfc             # .vue 文件解析
├── shared          # 共享代码

compiler

vue 编译相关的代码，包括将模板解析成 ast 语法树、ast 语法树的优化、生成 render 方法。 编译过程，可以在构建时，借助 webpack、vue-loader 等辅助工具进行编译。也可以在运行时，使用带有编译功能的 vue 进行编译。

core

vue 的核心代码，包括内置组件、全局 API 的封装、vue 的实例化、虚拟 DOM 等。

platforms

支持跨平台使用。可以使用在 web 客户端，也可以配合 weex 运行在 native 端。

server

支持服务端。可以在服务端把模板解析成 HTML 字符串。

sfc

vue 的单文件规范，会把 .vue 文件类型，解析成 JavaScript 对象。

shared

定义了一些 vue 中客户端和服务端公用的方法。
